# Offline AI Assistant

## Run the Project

### 1. Start Ollama

Make sure Ollama is running with the model:

```
ollama run llama3
```

(You only need to download the model once.)

### 2. Activate Virtual Environment

Windows:

```
venv\Scripts\activate
```

### 3. Run the Assistant

```
python main.py
```

The assistant will start in the terminal.

## Requirements

* Python 3.11+
* Ollama installed
* Llama3 model downloaded

## Install Dependencies

If needed, install packages using:

```
pip install -r requirements.txt
```
